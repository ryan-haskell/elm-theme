# elm-theme
> an experimental way to make a design system in Elm

## folder structure

```bash
- src/*                 # Elm package
- examples/01-intro/*   # Example usage
```

## try it out

1. Run `elm reactor`
2. Visit [http://localhost:8000/examples/01-intro/Main.elm](http://localhost:8000/examples/01-intro/Main.elm)
